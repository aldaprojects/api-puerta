Usuario

Crear cuenta

Se crea una cuenta con el rol 'USER_ROLE'

Metodo
    POST
URL
    https://api-puerta.herokuapp.com/usuario
Headers
    Ninguno
Params
    Ninguno
Body
  • name
  • email
  • password
  • username
  • fecha_registro

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "usuario": {
        "llave": false,
        "role": "USER_ROLE",
        "grupo": null,
        "google": false,
        "_id": EJEMPLO 5e48aad9b1f6d529a8ec382b,
        "name": EL NOMBRE DEL USUARIO,
        "email": EL CORREO DEL USUARIO,
        "username": EL USERNAME DEL USUARIO,
        "fecha_registro": EJEMPLO: 2020-01-27T00:00:00.000Z
    }
}

Iniciar sesión

Es necesario tener una cuenta ya creada para usar este método. Este generará un token que deberá usar posteriormente.

Metodo
    POST
URL
    https://api-puerta.herokuapp.com/login
Headers
    Ninguno
Params
    Ninguno
Body
  • email
  • password

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "usuario": {
        "llave": false,
        "role": "USER_ROLE",
        "grupo": null,
        "google": false,
        "_id": EJEMPLO 5e48aad9b1f6d529a8ec382b,
        "name": EL NOMBRE DEL USUARIO,
        "email": EL CORREO DEL USUARIO,
        "username": EL USERNAME DEL USUARIO,
        "fecha_registro": EJEMPLO 2020-01-27T00:00:00.000Z
    },
    "token": TOKEN
}
