Usuario

Crear cuenta

Se crea una cuenta con el rol 'USER_ROLE'. No require un token para usar.

Metodo
    POST
URL
    https://api-puerta.herokuapp.com/usuario
Headers
    Ninguno
Params
    Ninguno
Body
  • name
  • email
  • password
  • username
  • fecha_registro

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "usuario": {
        "llave": false,
        "role": "USER_ROLE",
        "grupo": null,
        "google": false,
        "_id": EJEMPLO 5e48aad9b1f6d529a8ec382b,
        "name": EL NOMBRE DEL USUARIO,
        "email": EL CORREO DEL USUARIO,
        "username": EL USERNAME DEL USUARIO,
        "fecha_registro": EJEMPLO: 2020-01-27T00:00:00.000Z
    }
}

Iniciar sesión

Es necesario tener una cuenta ya creada para usar este método. Este generará un token que deberá usar posteriormente. No requiere un token para usar.

Metodo
    POST
URL
    https://api-puerta.herokuapp.com/login
Token
    No
Headers
    Ninguno
Params
    Ninguno
Body
  • email
  • password

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "usuario": {
        "llave": false,
        "role": "USER_ROLE",
        "grupo": null,
        "google": false,
        "_id": EJEMPLO 5e48aad9b1f6d529a8ec382b,
        "name": EL NOMBRE DEL USUARIO,
        "email": EL CORREO DEL USUARIO,
        "username": EL USERNAME DEL USUARIO,
        "fecha_registro": EJEMPLO 2020-01-27T00:00:00.000Z
    },
    "token": TOKEN
}

Actualizar datos

Puede actualizar el nombre, nombre de usuario y la contraseña. Para usarse requiere enviar un body no vacio. Se requiere un token para usar.

Metodo
    PUT
URL
    https://api-puerta.herokuapp.com/usuario?id=[id]
Headers
    token
Params
    id
Body
  • email (opcional)
  • name (opcional)
  • password (opcional)

En caso de un status 200 como respuesta, obtendrá el nuevo usuario:


{
    "ok": true,
    "usuario": {
        "llave": false,
        "role": "USER_ROLE",
        "grupo": null,
        "google": false,
        "_id": EJEMPLO 5e48bc7b9c2efe3268c0ee9d,
        "name": NOMBRE ACTUALIZADO,
        "email": CORREO DEL USUARIO,
        "username": USERNAME ACTUALIZADO,
        "fecha_registro": EJEMPLO 2020-01-27T00:00:00.000Z
    }
}

Grupo

Crear grupo

Solo puede ser creado un grupo por usuario. Al crear el grupo, se cambiará el rol a 'ADMIN_ROLE', se asignará una llave al usuario y se actualizará su grupo. Se requiere un token para usar.

Metodo
    POST
URL
    https://api-puerta.herokuapp.com/grupo
Headers
    token
Params
    Ninguno
Body
  • name
  • id_admin (id del usuario que crea el grupo)
  • fecha_creacion

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "grupo": {
        "integrantes": [],
        "_id": EJEMPLO 5e49d1e664bcf30d8cbdfc26,
        "name": NOMBRE DEL GRUPO,
        "codigo": EJEMPLO ORKEK,
        "admin": EJEMPLO 5e48bc7b9c2efe3268c0ee9d,
        "fecha_creacion": EJEMPLO 2001-01-01T06:00:00.000Z,
        "__v": 0
    },
    "admin": {
        "llave": true,
        "role": "ADMIN_ROLE",
        "grupo": EJEMPLO 5e49d1e664bcf30d8cbdfc26,
        "google": false,
        "_id": EJEMPLO 5e48bc7b9c2efe3268c0ee9d,
        "name": NOMBRE DEL CREADOR,
        "email": CORREO DEL CREADOR,
        "username": USERNAME DEL CREADOR,
        "fecha_registro": EJEMPLO 2020-01-27T00:00:00.000Z
    }
}

Agregar integrante

Actualiza el Array de un grupo. Se requiere tener role 'ADMIN_ROLE'. Se requiere un token para usar.

Método
    PUT
URL
    https://api-puerta.herokuapp.com/grupo/[idGrupo]
    /integrante?id=[idIntegrante]
Headers
    token
Params
    id
Body
    Ninguno

En caso de un status 200 como respuesta, obtendrá:


{
    "ok": true,
    "grupo": {
        "integrantes": [
            {
                "_id": EJEMPLO 5e499cbc153539306864c3b7,
                "llave": false,
                "name": NOMBRE INTEGRANTE,
                "email": EMAIL INTEGRANTE,
                "username": USERNAME INTEGRANTE
            }
        ],
        "_id": EJEMPLO 5e49e37c80c93434a8317411,
        "name": NOMBRE DEL GRUPO,
        "codigo": CODIGO DEL GRUPO,
        "admin": EJEMPLO 5e48bc7b9c2efe3268c0ee9d,
        "fecha_creacion": EJEMPLO 2001-01-01T06:00:00.000Z,
        "__v": 0
    }
}
